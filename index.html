HTML

  • The head element contains information about the webpage not visible by the user.
  • The body element represents the visible content shown to the user.
  • HTML serves as the framework for a webpage.
  • It is the necessary scaffolding before style and logical functionality are introduced.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascasding Style Sheets.
  • It enablese the stylizing of several HTML elements simultaneously.
  • Using class selectors and class attributes, we can assign several properties to the same section elements in a markup.

Git

  • Git uses different commands to navigate github.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: pulls the most up to date version of the main branch.
  • git commit: commits any changes made to code to github.
  • git add: stages any changes made in the code.
  • Git add -> git commit -> git pull origin main -> git push origin "name of feature branch" is the general gitflow.
